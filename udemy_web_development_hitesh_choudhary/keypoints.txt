      |-> VS Code
      |-> Extension - Chai Theme
      |-> Settings - shortcut key 'Ctrl + Comma'
      |-> Extension - Live Server
      |-> Some Shortcuts in VS Code
          |-> To open Command Palette - Ctrl + Shift + P
          |-> Navigate between the files - Ctrl + P
          |-> Open Terminal - Ctrl + `
          |-> Toggle the side bars - Ctrl + B
          |-> Close current file - Ctrl + W
          |-> For saving file - Ctrl + S
          |-> To move cursor start or end of a line - Ctrl + Left or Right arrow key
          |-> Move cursor one word - Alt + Left or Right arrow
          |-> Select entire line - Ctrl + Shift
          |-> Duplicate line - Shift + Down arrow key
          |-> Remove line - Ctrl + X
          |-> To move line - Alt + Arrow key
          |-> Multi cursor - Alt + Click
