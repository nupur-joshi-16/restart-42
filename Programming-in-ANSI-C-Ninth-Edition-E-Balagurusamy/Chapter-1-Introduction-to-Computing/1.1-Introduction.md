# 1.1 : Introduction

__Definition of Computing:__<br>
Computing is an operation involving the processing of data through the use of a computer.

__Prerequisites for Computation:__<br>
To effectively use a computer for computation, one must:
* Be aware of the various hardware components within the system.
* Understand how these components function together to serve specific purposes.

__Chapter Objectives:__<br>
* Discuss hardware components.
* Explore program development methodologies.
* Address problem-solving and computational activities through developed programs.
